# COVID-19 Global Insights Dashboard
An interactive data analytics dashboard built with Python and Streamlit to visualize global COVID-19 trends.

### Features
- Analyzed **300,000+ rows** of global data.
- Static Insights:
    1 - Top affected countries by total cases
    2 - Global daily new cases trend
- Interactive Dashboard:
    1 - Country multi-select
    2 - Date range filtering
    3 - Vaccination progress visualization
- Comparison Table: Latest statistics for selected countries
- Export Option: Download cleaned dataset as CSV

### Tech Stack
- **Language:** Python
- **Libraries:** Pandas, Plotly, Streamlit
- **Data Source:** Our World in Data (OWID)

### How to Run
1. Clone the repo: `git clone [Your-Repo-Link]`
2. Install requirements: `pip install -r requirements.txt`
3. Run app: `streamlit run covid.py`
